# REST API Example

REST API example with NodeJS, MongoDB and Mongoose.

Credit to [Web Dev Simplified](https://youtu.be/DvlyzDZDEq4)

## Usage

### Install dependencies

```
yarn install
```

### Run app

```
yarn start
```

### Run dev mode

```
yarn start-dev
```

Spins up dev server.<br />
Make HTTP Requests to http://localhost:3010.
